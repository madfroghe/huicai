• 什么是账户总资产？

             账户总资产是指账户可用余额与当前投资金额的总和。

  • 什么是可用余额？

            可用余额是指当前账户中可用于提现、投资的资金数额。充值、投资、 投资产品到期回收本息均会造成账户可用余额的变动。

  • 如何查看我投资的理财产品？

           用户登录八锂米后，进入“我的账户”页面，可在“我的投资” 中查看当前未到期投资，可在“交易记录”中查看已到期投资。

  • 忘记登录密码怎么办？

            您可以点击“忘记密码”，通过注册手机号码和动态验证码重新设置登录密码并登录。

  • 如何绑定银行卡？

            您充值时使用的银行卡将默认绑定，可以通过“我的账户”-“账户管理”-“银行卡管理”添加或删除银行卡。

  • 为什么要绑定银行卡？

            绑定银行卡后，提现只能提取至已绑定的本人身份证开户的银行卡中，将极大降低资金被他人窃取风险。
